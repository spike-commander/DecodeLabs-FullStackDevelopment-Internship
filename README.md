## 🛠️ Core Engineering Philosophy

Every project committed to this repository follows rigid industry-standard mandates:
1. **Zero Framework Crutches:** Building from scratch using native web technologies (HTML5, CSS3, Vanilla JS) to master raw programmatic fundamentals before using abstractions.
2. **Mobile-First Infrastructure:** Designing systematically using fluid logic, explicit `min-width` media break points, and CSS mathematical properties rather than arbitrary static viewports.
3. **Semantic & Accessible (WCAG):** Authoring strict HTML structural landmark elements to ensure absolute integrity for screen readers, scrapers, and AI indexers.

### Milestone 1: The Responsive Architecture Layout
*   **Objective:** Create a fluid landing page layout mirroring modern, human-centric design tokens.
*   **Tech Stack:** Semantic HTML5, Native CSS3 (CSS Grid Macro structures + Flexbox Micro components), Vanilla JavaScript.

### Milestone 2: Backend API Development (The Nervous System)
*   **Objective:** Develop a reliable, lightweight Backend API using RESTful principles to process application logic and manage data flow.
*   **Tech Stack:** Node.js, Express.js, UUID, In-Memory Data Store.
*   **Key Attributes Met:**
    *   Stateless RESTful architecture with JSON data exchange.
    *   Two-layer validation (Syntactic + Semantic).
    *   Proper HTTP status codes (200, 201, 204, 400, 404, 500).
    *   CRUD operations: GET, POST, PUT, DELETE on `/items` resource.
